# miniMVC
one java web framework


##
mini framework 是一款模仿spring开发的web框架
功能包含
- mvc
- aop
- ioc
- 数据库事务实现
- 支持文件上传
- rest模块
- security模块
- soap模块
通过自己模仿开发一个属于自己的web框架我对一下技术有了新的认识
- java动态代理
- Java反射
